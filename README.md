# Universal Talent Source Website

Static marketing site for Universal Talent Source, ready to deploy on Vercel.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server.

## Vercel deployment

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Use the default static project settings.
4. Add `universaltalentsource.com` in Vercel Project Settings > Domains.
5. Update DNS records at the domain registrar using the values Vercel provides.

Before publishing, confirm the contact email in `index.html`.
